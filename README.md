# QA Self-learning – SauceDemo (Jira Hands-On)

## Overview
This is a **self-learning QA project** demonstrating hands-on experience with **Jira** for bug tracking, Agile workflows, filters, JQL queries, and dashboards.  
The project uses **SauceDemo**—a free demo e-commerce site intentionally designed with UI defects for practicing real QA workflows.

## Tools Used
- **Jira Software (Cloud)** – bug tracking, sprint board, filters, JQL, dashboards  
- **SauceDemo** – demo e-commerce site for QA practice
